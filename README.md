## 🚀 Overview

EcoSmart is a comprehensive **Smart Waste Management System** designed for smart cities and municipalities. It combines IoT sensor integration, machine learning algorithms, and real-time fleet tracking to optimize waste collection operations, reduce environmental impact, and cut operational costs by up to **30%**.

The system features a stunning dark-themed dashboard with glassmorphism UI, interactive data visualizations, and fully functional real-time simulations.

---

## ✨ Key Features

### 📊 Real-Time Monitoring
- **Live Bin Tracking** — Monitor fill levels, temperature, and status across all smart bins
- **Interactive Map Visualization** — Animated markers with color-coded status indicators (Normal/Warning/Critical)
- **Fleet GPS Tracking** — Real-time truck locations with route progress indicators
- **Critical Alerts** — Instant notifications for bins exceeding capacity thresholds

### 🤖 AI Route Optimization
- **Machine Learning Algorithms** — Automatically calculate optimal collection routes based on fill levels, traffic patterns, and historical data
- **Fuel Savings Calculator** — Track distance, time, and fuel savings vs. static routes
- **Dynamic Route Adjustment** — Real-time rerouting based on changing conditions
- **Predictive Analytics** — Forecast bin fill levels 24 hours in advance

### 📈 Advanced Analytics
- **Fill Level Trends** — Interactive Chart.js visualizations with time period filtering (7/30/365 days)
- **Waste Type Distribution** — Doughnut charts breaking down General, Recycling, Organic, and Hazardous waste
- **Route Efficiency Metrics** — Bar charts comparing performance across collection routes
- **Zone Volume Analysis** — Comparative waste generation by geographic zones
- **CO₂ Emissions Tracking** — Sustainability metrics and carbon offset calculations

### 🛠️ Operational Tools
- **Quick Actions Panel** — One-click route optimization, emergency dispatch, report generation
- **Bin Management** — Add, filter, and manage smart bins by zone and status
- **Fleet Management** — Track vehicle status (Collecting/En Route/Idle) with progress bars
- **Alert Center** — Centralized notification management with acknowledge/dismiss functionality
